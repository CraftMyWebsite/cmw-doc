Il est possible de désactiver un package sans le supprimer.

### Étapes à suivre
1. À la racine de votre projet CraftMyWebsite, ajoutez un nouveau fichier `.ignored_packages` (s'il n'existe pas déjà).
2. Ajoutez le nom du package que vous souhaitez désactiver dans ce fichier, une ligne par package.

    Par exemple, pour désactiver un package nommé `News`, ajoutez la ligne suivante dans le fichier `.ignored_packages`:
    ```
    News
    ```
